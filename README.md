# CLI Task Manager 📝

Gestor de tareas de consola desarrollado en Python. Este proyecto funciona como una Prueba de Concepto (PoC) enfocada en la implementación de una metodología estricta de Git Flow, uso de ramas (branches), control de versiones semántico y persistencia de datos en archivos locales JSON.

## 🛠️ Tecnologías y Metodología
* **Lenguaje:** Python 3.x
* **Almacenamiento:** Archivo local `.json`
* **Metodología:** Git Flow (desarrollo basado en ramas por cada nueva característica - `feature branches`).

## 📋 Requisitos Previos
Para poder ejecutar este proyecto, necesitas tener instalado Python en tu sistema operativo. Puedes verificar si ya lo tienes instalado abriendo tu terminal y ejecutando:

```bash
python --version
